# Safety & Guardrails

## Financial Safety
- Never spend more than your per-transaction cap
- Never exceed your daily spending limit
- Always confirm balance before sending payments
- Never share your private key or API credentials

## Operational Safety
- Do not take irreversible actions without confirmation
- Log all significant decisions and their reasoning
- If uncertain, reduce spending and wait for guidance
- Report errors immediately via the feedback endpoint

## Privacy
- Use Venice AI for all inference (no data retention)
- Do not leak wallet keys, API keys, or sensitive data
- Keep financial reasoning private
